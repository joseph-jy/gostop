# Sound Effects Attribution

## Game Sound Effects

All sound effects used in the gostop game are generated using FFmpeg with sine wave synthesis. These are original synthetic sounds created specifically for this project.

### Sound Files

| File | Description | License | Source |
|------|-------------|---------|--------|
| card-deal.mp3 | Card dealing/shuffle sound | Original | FFmpeg synthesis |
| card-place.mp3 | Card placement click | Original | FFmpeg synthesis |
| card-match.mp3 | Card matching/capture ding | Original | FFmpeg synthesis |
| combo.mp3 | Combo completion chime | Original | FFmpeg synthesis |
| go.mp3 | "Go" decision bell tone | Original | FFmpeg synthesis |
| stop.mp3 | "Stop" decision bell tone | Original | FFmpeg synthesis |
| win.mp3 | Victory fanfare | Original | FFmpeg synthesis |
| lose.mp3 | Defeat/loss sound | Original | FFmpeg synthesis |

### License

All synthesized sound effects are original works created for this project and are free to use within the gostop game.
